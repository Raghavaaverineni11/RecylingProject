# RecylingProject

A full-stack web application built for a university to record, track, and analyze daily waste and recycling activities. The system helps sustainability staff enter waste data, monitor recycling efficiency, and generate actionable reports through interactive dashboards.

The Recycling Waste Data Management Portal is an end-to-end application that allows authorized university staff to:

Enter daily waste logs and categorize items (recycled, dumped, compostable, etc.)
Track volumes of recycled vs dumped waste over time
Visualize insights using charts and dashboards
Generate reports for sustainability decision-making
Maintain historical waste data efficiently
This system improves the accuracy, transparency, and accessibility of campus waste management operations.
